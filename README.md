# Mary
태어나서 처음으로 만드는 로그라이크. 
First roguelike project ever.

## Long Term Goal
- 절차적 생성으로 생성된 엔티티
 - procedurally generated entities
- 절차적 생성으로 생성된 아이템
 - procedurally generated items
- 자유도 높고, 직관적이며, 간략한 아이템 조합
 - highly customized, intuitive yet simple item composition
